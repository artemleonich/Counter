# Counter

Простое iOS-приложение — счётчик с историей изменений.

## Возможности

- Увеличение и уменьшение значения счётчика
- Сброс значения
- История всех действий с временными метками
- Тактильная обратная связь при нажатии кнопок
- Защита от отрицательных значений

## Стек

- Swift, UIKit
- Storyboard
- Xcode 15+
- iOS 13+
